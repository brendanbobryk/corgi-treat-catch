A small, fun React game where you control a corgi and catch falling bone treats!  
This project is part of a daily game challenge to practice React fundamentals, basic game loops, and simple collision detection.

## Features

- Move the corgi left and right with arrow keys
- Bones fall from the top of the screen
- Catch bones to increase your score
- Smooth, continuous game loop
- Simple dark-themed layout with image assets

## Technologies Used

- React (with hooks)
- JavaScript
- CSS
- Vite

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/corgi-treat-catch.git

2. Navigate to the project folder:
    ```bash
    cd corgi-treat-catch

3. Install dependencies:
    ```bash
    npm install

4. Start the development server:
    ```bash
    npm run dev

5. Open your browser at the local address shown in the terminal (usually http://localhost:5173) to play!
